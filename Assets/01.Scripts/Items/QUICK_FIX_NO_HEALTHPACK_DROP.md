# 🎯 적이 힐팩 드롭 - 30초 해결!

## ❌ 문제
**적을 잡아도 힐팩이 나오지 않아요!**

## ✅ 해결 (30초!)

### 1단계: 힐팩 프리팹 확인 (5초)
- [ ] Project 창에 힐팩 프리팹이 있나요?
- [ ] 없다면 `QUICK_START_HEALTHPACK.md` 보고 먼저 만드세요!

### 2단계: 적 선택 (5초)
- [ ] Hierarchy에서 적 오브젝트 클릭
- [ ] 예: Snowman, BossSanta 등

### 3단계: 설정 변경 (20초)
1. Inspector에서 **Enemy Health** 컴포넌트 찾기
2. **Drop Settings** 섹션 찾기
3. **Drop Items** 배열 설정:
   ```
   Size: 1 입력
   Element 0: [힐팩 프리팹을 여기에 드래그!]
   ```
4. **Drop Chance**: `0.5` (50% 확률)

### 4단계: 테스트! (즉시)
- [ ] 플레이 모드 실행
- [ ] 적 죽이기
- [ ] 힐팩 나오는지 확인! 🎉

---

## 확률 설정

| 값 | 확률 | 설명 |
|----|------|------|
| 0.0 | 0% | 절대 안 나옴 |
| 0.2 | 20% | 5번 중 1번 |
| 0.5 | 50% | 2번 중 1번 |
| 1.0 | 100% | 무조건 나옴 (보스용) |

문제가 있다면:
- [ ] 힐팩 프리팹 만들었나요?
- [ ] Drop Items Size가 1 이상인가요?
- [ ] Element 0에 힐팩이 할당되었나요?
- [ ] Drop Chance가 0보다 큰가요?
- [ ] 적이 실제로 죽었나요?

## 프리팹 vs 씬 오브젝트

⭕ **올바름**: Project 창의 프리팹 드래그
```
Project/Prefabs/HealthPack → Drop Items
```

❌ **잘못됨**: Hierarchy의 오브젝트 드래그
```
Hierarchy/HealthPack → ❌ 안 됨!
```
